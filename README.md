# 🚀 Système de Gestion de Transferts d'Argent.   
## 📌 Description 
Ce projet est une application de console développée en langage C. Elle permet de gérer des opérations de transfert d'argent, incluant l'enregistrement des dépôts, les retraits, les annulations et le suivi des commissions.
L'application utilise une structure de données optimisée pour gérer jusqu'à 500 dépôts.
## ⚙️ Fonctionnalités
Le programme propose un menu complet pour:
- Gestion des transactions: Enregistrer, modifier, supprimer ou retirer un dépôt.
- Suivi financier: Calcul automatique des commissions et montant payable.
- Statistiques: Situation globale des dépots, retraits et commisions(nombre et montants totaux). 
- Filtres avancées: Lister les codes déposés, rétirés ou annulés par date précise ou entre deux dates.
- Sécurité: Accès au menu général protégé par une identification (Nom d'utilisateur et mot de passe).
